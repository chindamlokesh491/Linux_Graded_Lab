2. File Permissions

Before changing the permissions, the Documents folder and its files had the default permissions assigned by the system.

After using the chmod command, the Documents folder was assigned permission 700, allowing only the owner to access it. The project_plan.txt file was assigned permission 640, giving read and write access to the owner, read access to the group, and no access to others.

5. How Permissions Protect Project Data

Changing file permissions helps protect important project files from unauthorized access. Restricting access ensures that only the owner or authorized users can modify sensitive project documents, reducing the risk of accidental changes or deletion.
